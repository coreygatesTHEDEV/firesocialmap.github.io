What is Fire: The Social Map?

FireSocialMap helps you get the true vibe of a venue in real time — not the polished version, not the outdated reviews, but what people are actually experiencing right now. Through simple, crowd‑sourced signals, the map updates as the city moves, giving you an honest, up‑to‑the‑minute sense of where to go and what each place feels like. It’s built for last‑minute decisions and in‑the‑moment exploration, making it easy to see what’s happening around you at a glance.

Current & Upcoming Features

Below is a high‑level overview of the features included in the MVP and the next planned release.

Phase 0–1 Features (MVP)

Live Map Infrastructure

Core system for displaying real‑time activity pins.

Pin System

Markers representing venues or activities happening right now.

Voting System

Quick user feedback on activity levels and experiences.

Time‑Based Decay

Votes naturally fade to keep information fresh and relevant.

Optional Geo Indicators

Privacy‑safe, opt‑in signals showing general activity levels.

Analytics Instrumentation

Early usage metrics to support future improvements.

Phase 1.5 Features (MVP)

Community Profiles

Profiles for organizers, groups, and community partners.

Event Pins

Time‑bound pins for pop‑ups, meetups, and gatherings.

Venue Profiles

Optional enhanced profiles for venues.

Featured Pins

Non‑intrusive visibility boosts for venues and events.

Trend Insights

Early analytics for organizers and venues.
